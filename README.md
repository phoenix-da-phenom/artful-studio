# 🎨 Digital Art Studio

A collaborative platform for photographers, digital artists, illustrators, and creatives to connect, share ideas, collaborate on projects, and showcase their work.

Built with **Next.js 15 (App Router)**, TypeScript, and modern web technologies.

---

## ✨ Features

- **Real-time Collaboration** — Work together on mood boards, projects, and idea sessions
- **Project Workspaces** — Create and manage collaborative art projects
- **Feed & Discovery** — Browse inspiring work from the community
- **Idea Sharing** — Post concepts, sketches, references, and get feedback
- **User Profiles** — Showcase your portfolio and artistic journey
- **Comments & Discussions** — Engage in meaningful conversations around art
- **Asset Upload & Management** — Support for high-resolution images, videos, and files
- **Tagging & Categorization** — Easily discover work by style, medium, or theme
- **Dark Mode** — Beautiful interface optimized for creatives

---

## 🚀 Tech Stack

- **Framework**: Next.js 15 (App Router) + React 19
- **Language**: TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **Authentication**:Jwt
- **Database**: MongoDB
- **File Storage**: UploadThing 
- **Real-time**: Pusher / Socket.io
- **Deployment**: Vercel

## 🛠 Getting Started

### Prerequisites

- Node.js 20+
- pnpm / npm / yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/phoenix-da-phenom/artful-studio.git

# Navigate to the project directory
cd artful-studio

# Install dependencies
pnpm install
# or
npm install
# or
yarn install
